Pengenalan Front End dan Back End

Author : Alisa


Didalam pengembangan web, terdapat istilah front end dan back end. Front end developer bertanggung jawab dalam mengembangkan tampilan dan interaksi antara user
dengan website atau aplikasi. Sedangkan back end developer bertugas mengembangkan database dan logika pemrograman di balik layar.
